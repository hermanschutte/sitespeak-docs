---
title: 'Retrieve User Account'
api: 'GET https://api.sitespeak.ai/v1/me'
description: 'Retrieves your user account details and validates that your token is active'
---

## Headers

| Name | Type | Description |
|------|------|-------------|
| Accept* | String | application/json |
| Authorization* | String | Bearer {api_token} |

<RequestExample>
```bash cURL
curl -X GET https://api.sitespeak.ai/v1/me \
  -H "Accept: application/json" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.sitespeak.ai/v1/me', {
  method: 'GET',
  headers: {
    'Accept': 'application/json',
    'Authorization': 'Bearer YOUR_API_TOKEN'
  }
});

const data = await response.json();
console.log(data);
```
</RequestExample>
