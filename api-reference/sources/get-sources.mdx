---
title: 'Retrieve Sources'
api: 'GET https://api.sitespeak.ai/v1/{chatbot_id}/sources'
description: 'Returns the sources for the chatbot with their training statuses'
---

## Path Parameters

| Name | Type | Description |
|------|------|-------------|
| chatbot_id* | String | The ID of your chatbot. You can find the ID on the settings page for your chatbot. |

## Headers

| Name | Type | Description |
|------|------|-------------|
| Accept* | String | application/json |
| Authorization* | String | Bearer {api_token} |

<RequestExample>
```bash cURL
curl -X GET https://api.sitespeak.ai/v1/{chatbot_id}/sources \
  -H "Accept: application/json" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.sitespeak.ai/v1/{chatbot_id}/sources', {
  method: 'GET',
  headers: {
    'Accept': 'application/json',
    'Authorization': 'Bearer YOUR_API_TOKEN'
  }
});

const data = await response.json();
console.log(data);
```
</RequestExample>
