# URL Redirects

Use these redirects in Cloudflare Bulk Redirects to map old help center URLs to the new docs site.

## Format

| Source URL | Destination URL |
|------------|-----------------|

## Getting Started

| Source URL | Destination URL |
|------------|-----------------|
| `/help-center/getting-started/how-can-i-capture-leads-with-my-chatbot` | `/docs/getting-started/lead-capture` |
| `/help-center/getting-started/how-can-i-personalise-my-welcome-message` | `/docs/getting-started/personalise-welcome-message` |
